Transparent Background for Icons

22.Apr 2012 | Comments

Using xfce, most of the themes available customize the way your desktop icons look, but almost all of them add some kind of background to the icon label. If you like the background to be transparent, you need to create one file in your home folder.

Add the following to ~/.gtkrc-2.0 (create it if it does not exist)
