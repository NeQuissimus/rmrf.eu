Install Oracle 10g XE Under Ubuntu 10.04 X86

10.Sep 2010 | Comments

This is a short tutorial on how to install the free Oracle 10g XE under Ubuntu 10.04 x86.

  1. Create a swap file at least 1024 MB in size
  2. Add deb http://oss.oracle.com/debian unstable main non-free to /etc/apt/sources.list

  3. Retrieve repository key with

    sudo wget http://oss.oracle.com/el4/RPM-GPG-KEY-oracle -O- | sudo apt-key add - sudo apt-get update

  4. sudo apt-get install oracle-xe to install Oracle XE

  5. run sudo /etc/init.d/oracle-xe configure
  6. After the installation, add the code snippet below to your ~/.bash_aliases
  7. Run the sqlplus command with login sys as sysdba and the previously set password to connect to the database. Startup starts the database if it hasn’t been started automatically.

~/.bash_aliases

export ORACLE_HOME=/usr/lib/oracle/xe/app/oracle/product/10.2.0/server
export PATH=$PATH:$ORACLE_HOME/bin
export ORACLE_SID=XE
