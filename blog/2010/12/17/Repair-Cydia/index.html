Repair Cydia

17.Dec 2010 | Comments

Today I friend of mine told me that his Cydia would not work anymore. It did not even start up, so we had to fix it remotely. Luckily, he had openSSH installed on his phone…

Attention: Only download packages that end in “iphoneos-arm.deb”

  1. Go to http://apt.saurik.com/debs/ and keep it open
  2. Download the latest versions of berkeleydb, curl, apt7-key, apt7-ssl, apt7-lib and apt7
  3. Use an SCP client to push all those deb files from your computer to the iPhone
  4. Log into your iPhone via SSH
  5. Install each package with dpkg -i package.deb, replacing package with the file name (you will need to install the packages in the same order I listed them)
  6. Once you have successfully installed all packages, you can use this command: apt-get install â€“fix-broken which will fix all your broken packages.
  7. Now Cydia should work again
